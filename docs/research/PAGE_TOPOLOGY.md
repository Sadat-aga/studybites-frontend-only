# Studybites Login Page Topology

1. Full-viewport auth canvas on a pale gray background.
2. Top navigation with logo at left, language switcher + outlined CTA at right.
3. Centered blurred gradient backdrop positioned behind the auth card.
4. Elevated auth card with:
   - title
   - social auth row
   - divider
   - email field
   - password field with visibility toggle
   - forgot-password link
   - submit button
   - signup footer

Interaction model: mostly static with small click/hover-driven affordances.
